# Salon & Beauty Services Website

## About the Project
This is a simple and responsive salon website created to showcase beauty services and allow customers to book appointments easily.

## Services
- Haircut & Styling (Men, Women, Kids)
- Hair Spa, Coloring & Smoothening
- Facial, Cleanup & Skincare
- Waxing, Threading & Grooming
- Manicure, Pedicure & Nail Art
- Bridal & Party Makeup
- Home Service Option

## Target Customers
- Walk-in customers
- Working professionals
- Brides & event clients
- Membership customers
